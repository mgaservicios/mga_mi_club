# Coding Standards - MiClub

## Principios

-   Priorizar simplicidad y mantenibilidad.
-   Evitar duplicación de lógica.
-   Mantener módulos desacoplados.
-   Favorecer componentes reutilizables.

## Convenciones

-   TypeScript estricto.
-   Nombres descriptivos y consistentes.
-   Una responsabilidad por componente.

## Arquitectura

-   Organización por funcionalidades (features).
-   Separar UI, lógica y acceso a datos.
-   Mantener el dominio independiente del framework.
